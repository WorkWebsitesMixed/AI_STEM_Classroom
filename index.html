Section 1

The 3 Prompts

Three precision-engineered prompts that drive the full Plan → Create → Assess loop. Copy each one directly into your AI tool of choice.

Phase 1

🏗️ The Architect — Lesson Planning

You are an IGCSE 0445 Design and Technology teacher at a bilingual Cambridge school in Colombia. Students are Grade 10, fluent English speakers. Generate a 60-minute lesson plan for Logic Gates (AND, OR, NOT) using a traffic light as the real-world example, aligned to Cambridge IGCSE 0445 syllabus objective 4.4. Include a technical vocabulary list of 10 key terms with precise definitions suitable for exam responses.

Phase 2

🔧 The Builder — Student Activity

Write C++ code for an ESP32 on Wokwi that simulates a pedestrian traffic light. The normal cycle is: red for 10 seconds, green for 8 seconds, yellow for 3 seconds. When the pedestrian button is pressed during the red phase, activate a pedestrian green signal for 6 seconds before resuming the normal cycle. Add comments in plain English explaining what each section does.

Phase 3

📋 The Evaluator — AI Co-Grading

You are a Design and Technology teacher using the rubric below to assess a student's work. For each criterion, state the grade using the scale provided (1, 1.3, 1.7, 2, 2.3, 2.7, 3, 3.3, 3.7, 4) and explain specifically what is missing and how the student could improve to reach a higher level. Section 3

The Rubric

Assessment Rubric — Traffic Light System

Generated during the workshop using your school's grading scale

Criterion 1 – 1.7 2 – 2.7 3 – 4
System Description Vague or incomplete description of how the system behaves Mostly clear description with some gaps in precision Precise, complete description referencing all conditions and timing
Justification No reasoning given for design decisions Some reasoning present but not linked to requirements Fully justified with clear reference to system function and specification
Testing Evidence No testing mentioned Basic testing described ("it worked") Systematic testing described with specific conditions and results
💡 Adapt this rubric: replace the grading scale and criteria with your own subject's requirements. The AI prompt that generated this rubric is in Section 3.
